## SETUP (5 MINUTES)

### Desktop (Windows / Mac / Linux)
1. Download [SentinelGuard-v1.1.zip](https://github.com/fair-collective/sentinel-guard/releases/download/v1.1/SentinelGuard-v1.1.zip)  
2. Unzip  
3. Run `setup.exe` (Win) or `setup.sh` (Mac/Linux)  
4. Open `http://localhost:8000`  
5. Connect accounts via OAuth  

### Mobile (Android / iOS)
1. Install APK (Android) or TestFlight (iOS)  
2. Connect to desktop backend (`http://YOUR_IP:8000`)  
3. Use voice: "Sentinel, I'm traveling to Paris"  

### Browser
1. Load `extension/` in Chrome/Firefox  
2. Works alongside desktop/mobile
